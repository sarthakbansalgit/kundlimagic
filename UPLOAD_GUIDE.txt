## ðŸš¨ TROUBLESHOOTING

### Issue: White Screen / Blank Page

**Solution:**
1. Set folder permissions to 777:
   - application/cache/
   - application/logs/
   - application/sessions/

2. Check if `.htaccess` uploaded correctly

3. View error logs:
   - Download `application/logs/` folder
   - Check for error files

### Issue: 500 Internal Server Error

**Solution:**
1. Check `.htaccess` syntax
2. Verify PHP version is 7.4+ (check cPanel)
3. Check file permissions (755 or 777)
4. View server error logs in cPanel

### Issue: CSS/Images Not Loading

**Solution:**
1. Check `base_url` in `config.php` ends with `/`
2. Clear browser cache (Ctrl+Shift+R or Cmd+Shift+R)
3. Check if `assets/` folder uploaded correctly
4. Check browser console (F12) for errors

### Issue: Database Connection Error

**Solution:**
Your credentials are already set correctly:
```php
'hostname' => 'localhost',
'username' => 'studokch_astro',
'password' => '@Navneet123N',
'database' => 'studokch_astro',
```

If still error:
1. Verify database exists in phpMyAdmin
2. Verify user has ALL PRIVILEGES
3. Check if password has special characters (it does: `@`)
4. Test database connection in phpMyAdmin

### Issue: 404 Error on Pages

**Solution:**
1. Check if `.htaccess` exists in root
2. Check if mod_rewrite is enabled (ask hosting)
3. Update `config.php`:
   ```php
   $config['index_page'] = '';  // Should be empty
   ```
